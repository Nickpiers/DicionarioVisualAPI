# DicionarioVisualAPI
Backend do projeto de dicionário visual


### Pré-requisitos

* Node.js (v18 ou superior recomendado)
* npm 

### Instalação e Configuração

```
npm install
cp .env.example .env 
prisma migrate dev
npm run dev
```

### Insomnia

Importe o arquivo `/insomnia/insomnia_workspace` para ter acesso aos endpoints disponíveis.
